Definition 3.2.4. Identity matrix.

The \(n\times n\) identity matrix, denoted \(I_n\text{,}\) is the square \(n\times n\) matrix
\begin{equation*} I_n=\begin{bmatrix}1\amp 0\amp 0 \amp \dots\amp 0\\ 0\amp 1 \amp 0 \amp \dots \amp 0\\ 0\amp 0\amp 1 \amp \dots \amp 0\\ \vdots \amp \vdots \amp \amp \ddots \amp \vdots\\ 0\amp 0\amp \dots \amp 0\amp 1\end{bmatrix} \end{equation*}
with ones along the diagonal and zeros everywhere else. In other words, for all \(1\leq i\leq n\) and \(1\leq j\leq n\text{,}\) we have
\begin{equation*} (I_n)_{ij}=\begin{cases} 1 \amp \text{if } i=j \\ 0\amp \text{if } i\ne j \end{cases}\text{.} \end{equation*}
When there is no confusion about which dimension indentity matrix is intended, we will often write simply \(I\) instead of \(I_n\text{.}\)
