Corollary 4.4.16. Street smarts.

Let \(V\) be a vector space of dimension \(n\text{,}\) and let \(S\subseteq V\) be a subset.
  1. If \(\val{S} < n\text{,}\) then \(S\) does not span \(V\text{.}\)
  2. If \(\val{S}>n\text{,}\) then \(S\) is linearly dependent.
  3. If \(\val{S}=n\text{,}\) then the following are equivalent:
    1. The set \(S\) is a basis.
    2. The set \(S\) spans \(V\text{.}\)
    3. The set \(S\) is linearly independent.

Proof.

Statements (1) and (2) follow directly from Theorem 4.4.11. Statement (3) is an easy consequence of Theorem 4.4.15. For example, if \(S\) spans \(V\text{,}\) then there is a subset \(S'\) of \(S\) that is a basis of \(V\text{;}\) since all bases of \(V\) have \(n\) elements, and since \(\val{S}=n\text{,}\) we must have \(S'=S\text{;}\) thus \(S\) is a basis. The proof for a linear independent set \(S\) is similar, and is left to the reader.
