Corollary 4.1.16. Null space and linear systems.

Let \(A\in M_{mn}\) and \(\boldb\in \R^m\text{,}\) and suppose the linear system \(A\boldx=\boldb\) is consistent.
  1. There is a unique solution to the system if and only if \(\NS A=\{\boldzero\}\text{:}\) i.e., if and only if the only solution to \(A\boldx=\boldzero\) is the trivial one \(\boldx=\boldzero\text{.}\)
  2. There are infinitely many solutions if and only if there is a nonzero solution to \(A\boldx=\boldzero\text{.}\)
