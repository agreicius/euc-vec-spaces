Theorem 4.4.11. Basis bounds.

Let \(B\) be a basis of the vector space \(V\text{,}\) and suppose \(\val{B}=n\)
  1. If \(S\) spans \(V\text{,}\) then \(\val{S}\geq n\text{.}\)
  2. If \(S\) is linearly independent, then \(\val{S}\leq n\text{.}\)
  3. If \(B'\) is a basis for \(V\text{,}\) then \(\val{B'}=n\text{.}\)

Proof.

  1. Suppose by contradiction that \(\Span S=V\) and \(\val{S} < n\text{.}\) Then Theorem 4.4.10 would imply \(B\) is linearly dependent. Since this is a contradiction, we conclude that \(\val{S}\geq n\text{.}\)
  2. This also follows from Theorem 4.4.10: since \(B\) is a spanning set of \(V\) any set containing more than \(n=\val{B}\) elements must be linearly dependent.
  3. This follows from (1) and (2): if \(B'\) is a basis, then since \(B'\) spans, we have \(\val{B'}\geq n\) (1); and since \(B\) is linearly independent we have \(\val{B'}\leq n\) (2). We conclude that \(\val{B'}=n\text{.}\)
