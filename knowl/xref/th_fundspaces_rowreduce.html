Theorem 4.5.7. Fundamental spaces and row operations.

Let \(A\) be an \(m\times n\) matrix, and suppose \(A\) is row equivalent to \(B\text{.}\) The fundamental spaces of \(A\) and \(B\) are related as follows.
  1. \(\NS A=\NS B\text{.}\)
  2. \(\RS A=\RS B\text{.}\)
  3. In general \(\CS A \) is not equal to \(\CS B\text{.}\) However, we do have
    \begin{equation*} \dim \CS A=\dim \CS B\text{.} \end{equation*}
    In fact, letting \(\bolda_i, \boldb_i\in \R^m\) be the columns of \(A\) and \(B\text{,}\) respectively, the columns \(\bolda_{i_1}, \bolda_{i_2},\dots, \bolda_{i_r}\) form a basis of \(\CS A\) if and only if the columns \(\boldb_{i_1}, \boldb_{i_2},\dots, \boldb_{i_r}\) form a basis of \(\CS B\text{.}\)

Proof.

Assume that \(A\) is row equivalent to \(B\text{.}\) Using Corollary 3.4.17, we see that this means there is an invertible matrix \(Q\) such that \(B=QA\text{,}\) and hence also \(A=Q^{-1}B\text{.}\) We will make use of this matrix \(Q\) below.
  1. The fact that \(\NS A=\NS B\) is a consequence of Theorem 2.1.16, since if \(A\) is row equivalent to \(B\text{,}\) then so are the linear systems with augmented matrices \([A\vert\boldzero]\) and \([B\vert\boldzero]\text{.}\)
  2. First we show that \(\RS B\subseteq \RS A\text{.}\) Since \(\RS B\) is defined as the span of the rows of \(B\text{,}\) by Theorem 4.2.6 it is enough to show that each row of \(B\) is an element of \(\RS A\text{.}\) For all \(1\leq i\leq m\text{,}\) let \(\bolda_i, \boldb_i, \boldq_i\) denote the \(i\)th rows of \(A\text{,}\) \(B\text{,}\) and \(Q\text{,}\) respectively, treated as row vectors. Applying Theorem 3.1.29 to the matrix product \(B=QA\text{,}\) we have \(\underset{1\times n}{\boldb_i}=\underset{1\times n}{\boldq_i} A\) for all \(1\leq i\leq m\text{.}\) Furthermore, using Theorem 3.1.29 again, the row vector \(\underset{1\times n}{\boldb_i}=\underset{1\times n}{\boldq_i} A\) is itself a linear combination of the rows \(\bolda_i\)of \(A\text{.}\) By definition of row space, this means \(\boldb_i\in \RS A\) for all \(1\leq i\leq m\text{,}\) and hence \(\boldr_i\in\RS A\text{,}\) as desired.
    It remains to show that \(\RS A\subseteq \RS B\text{.}\) But this follows using the same argument as above, using the fact that \(A=Q^{-1}B\text{:}\) i.e., by swapping the roles of \(A\) and \(B\text{,}\) and replacing \(Q\) with \(Q^{-1}\text{.}\)
  3. We now let \(\bolda_j, \boldb_j\) be the columns of \(A\) and \(B\text{,}\) respectively, where \(1\leq j\leq n\text{.}\) Since \(\CS A=\Span\{\bolda_1,\bolda_2,\dots, \bolda_n\}\text{,}\) by Theorem 4.4.15 there is a subset of columns that forms a basis of \(\CS A\text{.}\) Let \(S=\{\bolda_{i_1}, \bolda_{i_2},\dots, \bolda_{i_r}\}\) be any such a basis, so that \(\dim \CS A=r\text{.}\) We claim that the corresponding set of columns of \(S'=\{\boldb_{i_1},\boldb_{i_2},\dots, \boldb_{i_r}\}\) is a basis of \(\CS B\text{,}\) in which case \(\dim \CS B=\dim \CS A=r\text{.}\) To do so, we will use Theorem 4.3.4: in more detail, we will show that given any \(\boldw\in \CS B\text{,}\) we can write
    \begin{equation*} \boldw=c_{i_1}\boldb_{i_1}+c_{i_2}\boldb_{i_2}+\cdots +c_{i_r}\boldb_{i_r}\text{,} \end{equation*}
    in a unique way. Indeed, we have
    \begin{align*} \boldw\in \CS B \amp \iff \boldw\in \CS QA \amp (B=QA)\\ \amp \iff \boldw=QA\boldx \text{ for some } \boldx\in \R^n\\ \amp \iff Q^{-1}\boldw=A\boldx \text{ for some } \boldx\in \R^n\\ \amp \iff Q^{-1}\boldw\in \CS A\\ \amp \iff Q^{-1}\boldw=c_{i_1}\bolda_{i_1}+\cdots +c_{i_r}\bolda_{i_r} \end{align*}
    for a unique \(r\)-tuple \((c_{i_1}, c_{i_2},\dots, c_{i_r})\text{.}\) Here we have used the fact that the subset \(\{\bolda_{i_1},\bolda_{i_2},\dots, \bolda_{i_r}\}\) is a basis of \(\CS A\text{.}\) But this means
    \begin{align*} \boldw \amp =Q(c_{i_1}\bolda_{i_1}+\cdots +c_{i_r}\bolda_{i_r})\\ \amp = c_{i_1}Q\bolda_{i_1}+\cdots +c_{i_r}Q\bolda_{i_r}\\ \amp =c_{i_1}\boldb_{i_1}+\cdots +c_{i_r}\boldb_{i_r}\text{,} \end{align*}
    since \(\boldb_{i}=Q\bolda_i\) for all \(1\leq i\leq n\) by Theorem 3.1.27. Furthermore, the coefficients in the linear combination
    \begin{equation*} \boldw=c_{i_1}\boldb_{i_1}+\cdots +c_{i_r}\boldb_{i_r} \end{equation*}
    are unique, since our work above shows that
    \begin{equation*} \boldw=c_{i_1}\boldb_{i_1}+\cdots +c_{i_r}\boldb_{i_r} \iff Q^{-1}\boldw=c_{i_1}\bolda_{i_1}+\cdots +c_{i_r}\bolda_{i_r}\text{.} \end{equation*}
