Procedure 4.2.15. Linear independence of a finite set.

Let \(V\) be a vector space, and let \(S=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\subseteq V\text{,}\) where the \(\boldv_i\) are distinct.
  1. Write out the general vector equation
    \begin{equation*} c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n=\boldzero\text{.} \end{equation*}
  2. Translate this vector equation into a homogeneous linear system in the unknowns \(c_1,c_2,\dots, c_n \text{,}\) using the definition of equality for your vector space.
  3. Decide, using Gaussian elimination, whether this system has any nonzero (i.e., nontrivial) solutions. If it has no nontrivial solution, \(S\) is linearly independent; if it has a nontrivial solution, \(S\) is linearly dependent.
