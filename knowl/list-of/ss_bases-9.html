Example 4.3.6. Basis for \(\R_{>0}\).

Let \(V=\R_{>0}\text{,}\) and let \(S=\{\boldv\}\text{,}\) where \(\boldv=a\) is any positive real number. Prove: \(S\) is a basis if and only if \(a\ne 1\text{.}\)
Solution.
\((a=1\implies S \text{ not a basis})\).
Suppose \(a=1\text{.}\) Since \(1=\boldzero\in V\text{,}\) we have \(S=\{\boldzero\}\text{.}\) Any set containing the zero vector is linearly dependent ([cross-reference to target(s) "rm_linear_independence" missing or not unique]). Thus \(S\) is not a basis.
\((a\ne 1 \implies S \text{ is a basis})\).
Since \(S=\{\boldv\}\) consists of one nonzero element, it is linearly independent ([cross-reference to target(s) "rm_linear_independence" missing or not unique]). It remains only to show that \(S\) spans \(\R_{>0}\text{,}\) which amounts to showing that every \(b\in \R_{>0}\) is a scalar multiple of \(\boldv=a\text{.}\) Since by definition scalar multiplication in \(\R_{>0}\) is defined as \(c\boldv=a^c\text{,}\) this is equivalent to showing that every \(b\in \R_{>0}\) can be written in the form \(b=a^c\text{.}\) This fact is a familiar result from calculus, where you learn that the range (or image) of any exponential function \(f(x)=a^x\) is the set of all positive real numbers.
