Theorem 4.2.17. Invertibility theorem.

Let \(A\) be an \(n\times n\) matrix. The following statements are equivalent.
  1. \(A\) is invertible.
  2. The matrix equation
    \begin{equation*} A\underset{n\times 1}{\boldx}=\underset{n\times 1}{\boldb} \end{equation*}
    has a unique solution for any \(\boldb\in \R^n\text{.}\)
  3. The matrix equation
    \begin{equation*} A\underset{n\times 1}{\boldx}=\underset{n\times 1}{\boldb} \end{equation*}
    has a solution for any \(\boldb\in \R^n\text{.}\)
  4. The matrix equation
    \begin{equation*} A\underset{n\times 1}{\boldx}=\underset{n\times 1}{\boldzero} \end{equation*}
    has a unique solution: namely, \(\boldx=\boldzero_{n\times 1}\text{.}\)
  5. \(A\) is row equivalent to \(I_n\text{,}\) the \(n\times n\) identity matrix.
  6. \(A\) is a product of elementary matrices.
  7. \(\det A\ne 0\text{.}\)
  8. The set of columns of \(A\) spans \(\R^n\text{.}\)
  9. The set of columns of \(A\) is linearly independent.
  10. The set of rows of \(A\) spans \(\R^n\text{.}\)
  11. The set of rows of \(A\) is linearly independent.

Proof.

Statements 8–9, can be shown to be equivalent to one of the previous statements of the invertibilty theorem using the column method 3.1.27 of matrix multiplication. Indeed, given \(\boldb\in \R^n\) the matrix equation \(A\boldx=\boldb\) has a solution if and only if \(\boldb\) can be written as a linear combination of the columns of \(A\text{.}\) Thus, the matrix equation \(A\boldx=\boldb\) has a solution for all \(\boldb\in \R^n\) if and only if all vectors \(\boldb\in \R^n\) lie in the span of the rows of \(A\text{.}\) This proves statement 8 is equivalent to statement 3. Similarly, the matrix equation \(A\boldx=\boldzero\) has a nontrivial solution if and only if there is a nontrivial linear combination of the columns of \(A\) equal to \(\boldzero\text{,}\) if and only if the columns of \(A\) are linearly independent. This proves statement 9 is equivalent to statement 4.
Statements 8–10 are easily seen to be equivalent to statements 8–10 using the fact that \(A\) is invertible if and only if \(A^T\) is invertible.
