Theorem 3.1.16. Vector space structure of \(M_{mn}\).

Let \(m\) and \(n\) be positive integers.
  1. The set \(M_{mn}\text{,}\) together with the matrix addition and scalar multiplication, is a vector space.
  2. The zero vector of \(M_{mn}\) is the \(m\times n\) zero matrix \(\boldzero_{m\times n}\text{.}\)
  3. Given a matrix \(A\in M_{mn}\) its vector inverse is the additive inverse matrix \(-A\text{.}\)

Proof.

Having identified the relevant operations, as well as the proposed zero vector and vector inverses, we must verify that the vector axioms hold. We do so for Axioms i, iii, and iv, leaving the rest as an exercise.
