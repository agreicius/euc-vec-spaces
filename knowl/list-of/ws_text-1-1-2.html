Definition 4.5.1.
Let \(X\) be a topological space.
A set \(A\subseteq X\) is {\bf maximally connected} if \(A\) is connected, and for all connected sets \(B\text{,}\) if \(A\subseteq B\text{,}\) then \(A=B\text{.}\)
Similarly, a set \(A\subseteq X\) is maximally path connected if \(A\) is path connected, and for all path connected sets \(B\text{,}\) if \(A\subseteq B\text{,}\) then \(A=B\text{.}\)
Given \(x\in X\) its {\bf connected component} \(C(x)\) is the union of all connected sets containing \(x\text{;}\) similarly, its {\bf path-connected component} \(C_p(x)\) is the union of all path-connected sets containing \(x\text{.}\)
