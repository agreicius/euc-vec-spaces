Example 1.1.8. Zero vector space.

Let \(V=\{\bot\}\text{,}\) a set containing exactly one element. Show that \(V\) has a unique vector space structure, with respect to which \(\bot\) must be the zero vector of \(V\text{,}\) and hence that \(V=\{\bot\}=\{\boldzero\}\text{.}\)
Solution.
Since \(\bot\) is the only element of \(V\text{,}\) we must define vector addition and scalar multiplication as follows:
\begin{align*} \bot+\bot \amp = \bot\\ c\cdot \bot \amp = \bot \text{ for all } c\in \R\text{.} \end{align*}
It is then easy to see that \(V\) together with these operations constitutes a vector space.
Indeed, starting with axiom (iii), since \(V\) contains only one element, our only possible choice for the zero vector of \(V\) is \(\boldzero=\bot\text{.}\) Furthermore, this choice clearly satisfies (1.1.1), since for all \(\boldv\in V\) we have \(\boldv=\bot\) (since \(V=\{\bot\}\)) and thus
\begin{align*} \bot+\boldv \amp =\bot+\bot\\ \amp = \bot\\ \amp =\boldv\text{.} \end{align*}
Similarly, to show that vector inverses exist (axiom (iv)) amounts to showing that \(\bot\) has a vector inverse. Defining \(-\bot=\bot\text{,}\) we see that
\begin{align*} -\bot+\bot \amp = \bot+\bot\\ \amp = \bot\\ \amp = \boldzero\text{,} \end{align*}
since \(\bot=\boldzero\text{,}\) as we have already determined.
Similarly, the identities of axioms (i)-(ii) and (v)-(viii) are easy to verify, and amount to trivial statements. Consider axiom (vii), for example. For all \(\boldv,\boldw\in V\text{,}\) we have \(\boldv=\boldw=\bot\text{,}\) in which case
\begin{align*} c(\boldv+\boldw) \amp =c(\bot+\bot) \\ \amp =c\cdot \bot\\ \amp = \bot \amp \text{(def. of sc. mult.)} \end{align*}
and
\begin{align*} c\boldv+d\boldw \amp =c\cdot \bot+d\cdot \bot\\ \amp = \bot+\bot\\ \amp =\bot\text{.} \end{align*}
Thus
\begin{equation*} c(\boldv+\boldw)=\bot=c\boldv+d\boldw \end{equation*}
for all \(\boldv, \boldw\in V\text{.}\)
