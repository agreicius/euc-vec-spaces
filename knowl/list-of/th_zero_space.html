Definition 1.1.9. Zero vector space.

Let \(V=\{\bot\}\text{,}\) a set containing exactly one element. There is a unique vector space structure that can be defined for \(V\text{,}\) given as follows.
  • Vector operations.
    Vector addition on \(V\) is defined as \(\bot+\bot=\bot\text{;}\) scalar multiplication on \(V\) is defined as \(c\cdot \bot=\bot\text{.}\)
  • Zero vector.
    The zero vector of \(V\) is \(\bot\text{:}\) i.e., \(\boldzero=\bot\text{.}\)
  • Vector inverses.
    The vector inverse of \(\bot\) is \(\bot\text{:}\) i.e., \(-\bot=\bot\text{.}\)
Since \(\bot=\boldzero\) with respect to this vector space structure, we have \(V=\{\boldzero\}\text{.}\) We call \(V\) a zero space.
