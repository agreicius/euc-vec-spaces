Let \(A= \{ 2, 3, 6, 7, 8 \}\) , \(B= \{ 2, 4, 7, 8 \}\) , \(C= \{ 0, 1, 4, 5, 6, 7 \}\text{.}\)
List the elements of the following sets. If there is more than one element write them separated by commas.
\(A \cap B = \{\) \(\}\)
\(A \cup B = \{\) \(\}\)
\((B \cup C) \cap A = \{\) \(\}\)
\(B \cup (C \cap A) = \{\) \(\}\)
Answer 1.
\(2, 7, 8\)
Answer 2.
\(2, 3, 4, 6, 7, 8\)
Answer 3.
\(2, 6, 7, 8\)
Answer 4.
\(2, 4, 6, 7, 8\)
