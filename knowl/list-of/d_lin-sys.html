Definition 2.1.9. Systems of linear equations.

A system of linear equations (or linear system) is a set of linear equations.

The linear system (2.1.4) is homogeneous if each of its equations is homogeneous: i.e., if \(b_i=0\) for all \(1\leq i\leq m\text{.}\)
A solution to the system (2.1.4) is an \(n\)-tuple \((s_1,s_2,\dots,s_n)\) that is a solution to each of the systemâ€™s \(m\) equations. We say \((s_1,s_2,\dots, s_n)\) solves the system in this case.
A linear system is consistent if a solution to the system exists, and inconsistent otherwise.
