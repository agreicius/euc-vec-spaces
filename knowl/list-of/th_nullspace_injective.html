Theorem 5.2.12. Null space and injectivity.

Let \(T\colon V\rightarrow W\) be a linear transformation.
  1. Given vectors \(\boldv, \boldv'\in V\text{,}\) we have
    \begin{equation} T(\boldv)=T(\boldv') \iff \boldv'=\boldv+\boldu \text{ for some } \boldu\in \NS T\text{.}\tag{5.2.3} \end{equation}
    Equivalently, given \(\boldw\in W\) and \(\boldv\in V\) satisfying \(T(\boldv)=\boldw\text{,}\) the set \(X\) of all \(\boldv'\) satisfying \(T(\boldv')=\boldw\) can be described as
    \begin{equation} X=\{\boldv+\boldu\colon \boldu\in \NS T\}\text{.}\tag{5.2.4} \end{equation}
  2. We have
    \begin{equation*} T\text{ injective}\iff \NS T=\{\boldzero_V\}\text{.} \end{equation*}

Proof.

  1. We have
    \begin{align*} T(\boldv)=T(\boldv') \amp \iff T(\boldv')-T(\boldv)=\boldzero\\ \amp \iff T(\boldv'-\boldv)=\boldzero \amp (T \text{ is linear})\\ \amp \iff \boldu=\boldv'-\boldv\in\NS T \amp (\text{def. } \NS T)\\ \amp \iff \boldv'=\boldv+\boldu \text{ for some } \boldu\in\NS T \text{.} \end{align*}
    Equation (5.2.4) follows directly from (5.2.3) by observing that if \(T(\boldv)=\boldw\text{,}\) then \(T(\boldv')=\boldw\) if and only if \(T(\boldv)=T(\boldv')\text{.}\)
  2. According to (5.2.3) we have \(T(\boldv)=T(\boldv')\) if and only if \(\boldv'=\boldv+\boldu\) for some \(\boldu\in \NS T\text{.}\)
    If \(\NS T=\{\boldzero_V\}\text{,}\) then \(T(\boldv)=T(\boldv')\) implies \(\boldv'=\boldv+\boldzero=\boldv\text{.}\) Thus \(T\) is injective in this case.
    Conversely, if \(\NS T\ne \{\boldzero_V\}\) we can find a nonzero \(\boldu\in \NS T\text{.}\) It follows that for any \(\boldv\in V\) we have \(T(\boldv)=T(\boldv+\boldu)\text{.}\) Furthermore, since \(\boldu\ne\boldzero_V\text{,}\) we have \(\boldv\ne \boldv+\boldu\text{.}\) Thus \(T\) is not injective in this case.
