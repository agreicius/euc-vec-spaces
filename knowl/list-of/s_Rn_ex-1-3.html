Exercise 1. Positive reals.
Define \(\R_{> 0}\) to be the set of all positive real numbers: i.e.,
\begin{equation*} \R_{> 0}=\{r\in \R\colon r> 0\}\text{.} \end{equation*}
Define vector addition on \(\R_{> 0}\) to be real number multiplication, and define scalar multiplication on \(\R_{> 0}\) to be real number exponentiation: i.e., given vectors \(\boldv=r\) and \(\boldw=s\text{,}\) and scalar \(k\text{,}\) we define
\begin{align*} \boldv\oplus\boldw \amp = r\, s\\ k\odot\boldv \amp = r^k \text{.} \end{align*}
