Example 5.1.9. Elementary linear transformation proofs.

Let \(V\) and \(W\) be vector spaces.
  1. Prove that the zero transformation \(T_0\colon V\rightarrow W\) is a linear transformation.
  2. Prove that the identity transformation \(\id\colon V\rightarrow V\) is a linear transformation.
Solution.
  1. Let \(T_0\colon V\rightarrow W\) be the zero function: i.e., \(T(\boldv)=\boldzero\) for all \(\boldv\in V\text{.}\) We verify each defining property separately.
    1. Given \(\boldv, \boldw\in V\text{,}\) we have
      \begin{align*} T_0(\boldv_1+\boldv_2)\amp =\boldzero_W \amp (\text{by def.}) \\ \amp =\boldzero_W+\boldzero_W \\ \amp = T_0(\boldv_1)+T_0(\boldv_2) \amp (\text{by def.})\text{.} \end{align*}
    2. Given \(c\in \R\) and \(\boldv\in V\text{,}\) we have
      \begin{align*} T_0(c\boldv) \amp = \boldzero_W \amp (\text{def. of } T_0)\\ \amp = c\boldzero_W \amp (\knowl{./knowl/xref/th_vectorspace_props.html}{\text{1.1.9}} ) \\ \amp = cT_0(\boldv) \amp (\text{def. of } T_0)\text{.} \end{align*}
    This proves that \(T_0\colon V\rightarrow W\) is a linear transformation.
  2. Let \(\id\colon V\rightarrow V\) be the identity function: i.e., \(\id(\boldv)=\boldv\) for all \(\boldv\in V\text{.}\)
    1. Given \(\boldv, \boldw\in V\text{,}\) we have
      \begin{align*} \id(\boldv_1+\boldv_2)\amp =\boldv_1+\boldv_2 \amp (\text{def. of } \id)\\ \amp = \id(\boldv_1)+\id(\boldv_2) \amp (\text{def. of } \id)\text{.} \end{align*}
    2. Given \(c\in \R\) and \(\boldv\in V\text{,}\) we have
      \begin{align*} \id(c\boldv) \amp = c\boldv \amp (\text{def. of } \id)\\ \amp = c\id(\boldv) \amp (\text{def. of } \id) \text{.} \end{align*}
    This proves that \(\id\colon V\rightarrow V\) is a linear transformation.
