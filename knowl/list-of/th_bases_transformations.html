Theorem 5.1.21. Bases and linear transformations.

Let \(V\) and \(W\) be vector spaces, and let \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) be basis of \(V\text{,}\) where \(\boldv_i\ne \boldv_j\) for all \(i\ne j\text{.}\)
  1. Uniqueness of transformations.
    Given linear transformations \(T\) and \(T'\) from \(V\) to \(W\text{,}\) if \(T(\boldv_i)=T'(\boldv_i)\) for all \(1\leq i \leq n\text{,}\) then \(T=T'\text{.}\)
  2. Existence of transformations.
    Given any choice of vectors \(\boldw_1,\boldw_2,\dots, \boldw_n\in W\text{,}\) there is a unique linear transformation \(T\colon V\rightarrow W\) satisfying
    \begin{equation*} T(\boldv_i)=\boldw_i \end{equation*}
    for all \(1\leq i\leq n\text{.}\) In more detail, \(T\) is defined as follows: if \(\boldv\in V\) is written in terms of \(B\) as \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\) for scalars \(c_i\in \R\text{,}\) then
    \begin{equation} T(\boldv)=c_1\boldw_1+c_2\boldw_2+\cdots +c_n\boldw_n\text{.}\tag{5.1.1} \end{equation}

Proof.

Proof of (1).
Since any \(\boldv\in V\) has a unique expression of the form
\begin{equation*} \boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n \text{,} \end{equation*}
where \(c_i\ne 0\) for all \(1\leq i\leq r\text{,}\) the formula in (5.1.1) defines a function \(T\colon V\rightarrow W\) in a well-defined manner. Note also that the formula still applies even if some of the coefficients are equal to 0: if \(c_i=0\text{,}\) then \(c_iT(\boldv_i)=\boldzero\text{,}\) and the right-hand side of (5.1.1) is unchanged. We will use this fact below.
We now show that \(T\) is linear. Given \(\boldv, \boldv'\in V\) we can find a common collection of elements \(\boldv_1,\boldv_2,\dots, \boldv_n\in B\) for which
\begin{align*} \boldv \amp = c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\\ \boldv'\amp=d_1\boldv_1+d_2\boldv_2+\cdots +d_n\boldv_n \end{align*}
for some \(c_i, d_i\in \R\text{.}\) We can no longer assume that \(c_i\ne 0\) and \(d_i\ne 0\) for all \(1\leq i\leq r\text{,}\) but as observed above we still have
\begin{align*} T(\boldv) \amp = c_1T(\boldv_1)+c_2T(\boldv_2)+\cdots +c_nT(\boldv_n)\\ T(\boldv')\amp=d_1T(\boldv_1)+d_2T(\boldv_2)+\cdots +d_nT(\boldv_n) \text{.} \end{align*}
Given any \(c, d\in \R\text{,}\) we have
\begin{align*} T(c\boldv+d\boldv')\amp=T(cc_1\boldv_1+cc_2\boldv_2+\cdots +cc_n\boldv_n+dd_1\boldv_1+dd_2\boldv_2+\cdots +dd_n\boldv_n) \\ \amp= T\left((cc_1+dd_1)\boldv_1+(cc_2+dd_2)\boldv_2+\cdots +(cc_n+dd_n)\boldv_n\right) \\ \amp =(cc_1+dd_1)T(\boldv_1)+(cc_2+dd_2)T(\boldv_2)+\cdots +(cc_n+dd_n)T(\boldv_n) \amp (\knowl{./knowl/xref/eq_bases_transformations.html}{\text{(5.1.1)}} ) \\ \amp= c(c_1T(\boldv_1)+c_2T(\boldv_2)+\cdots +c_nT(\boldv_n))+d(d_1T(\boldv_1)+d_2T(\boldv_2)+\cdots +d_nT(\boldv_n)\\ \amp =cT(\boldv)+dT(\boldv')\text{.} \end{align*}
Thus \(T\) is a linear transformation.
Proof of (2).
Assume \(T\) and \(T'\) are linear transformations from \(V\) to \(W\) satisfying \(T(\boldv_i)=T'(\boldv_i)\) for all \(1\leq i\leq n\text{.}\) Given any \(\boldv\in V\) we can write \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\text{.}\) It follows that
\begin{align*} T(\boldv) \amp = T(c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n)\\ \amp = c_1T(\boldv_1)+c_2T(\boldv_2)+\cdots +c_nT(\boldv_n) \amp (T \text{ is linear})\\ \amp =c_1T'(\boldv_1)+c_2T'(\boldv_2)+\cdots +c_nT'(\boldv_n)\\ \amp = T'(c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n) \amp (T' \text{ is linear}) \\ \amp = T'(\boldv)\text{.} \end{align*}
Since \(T(\boldv)=T'(\boldv)\) for all \(\boldv\in V\text{,}\) we have \(T=T'\text{.}\)
